# produtoshotad

Pipeline de IA que transforma 1 foto de produto em 5 imagens de anúncio e 5 vídeos curtos (9:16), em 3 workers (Agnes/US$0, HyperFrames/sem IA de vídeo, ou o pipeline original Fal.ai/Seedance + ElevenLabs).

## 📖 Guia de uso

Guia completo (landing + passo a passo): **https://inematds.github.io/produtoshotad/guia/**

Ver `doc/CLAUDE.md` e `doc/PLANO-WORKFLOWS.md` para a documentação técnica completa dos 3 workers.
